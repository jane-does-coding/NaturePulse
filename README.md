## Run Locally

Clone the project

```bash
  git clone https://github.com/jane-does-coding/NaturePulse
```

Go to the project directory

```bash
  cd my-project
```

Install dependencies

```bash
  npm i
```

Start the server

```bash
  npm run dev
```
